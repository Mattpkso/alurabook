# 📚 Projeto: AluraBook

Interface de uma livraria digital chamada **AluraBook**, desenvolvida com foco em responsividade e design mobile-first.

## 🚀 Tecnologias utilizadas

- HTML  
- CSS  

## ✅ Funcionalidades

- Layout responsivo (mobile-first)  
- Menu interativo com abas e categorias usando apenas HTML e CSS  
- Seções de destaques, lançamentos e mais vendidos  
- Botões de ação para favoritos e carrinho (layout visual)
